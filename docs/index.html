Avaxcast

A prediction market for cryptocurrency pair prices, built on Avalanche & uses Chainlink price feeds to resolve markets.

Website: https://040bl7t9ja07qmi7an751hujghtqkhtsfr6n6j746dqljhmrocb4ds0.siasky.net/

Demo: https://youtu.be/OHCry_mTMYc

Skynet

We use Skynet to deploy our frontend via their Github Action.

To add our app to your Skynet Homescreen please press the button above.

We also use SkyDB to store a JSON containing an array of all deployed Prediction Market contract addresses
